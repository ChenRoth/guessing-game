The Guessing Game API

This site has an API for creating and playing the guessing game

How do you play the guessing game?

  • Create a game
  • Add at least 2 players to the game - each has a name and a guessing number between 1 and 100
  • Start the game and a random number is picked

The winner is the player who guessed the closest number to the random one!

How do you use the API?

  • GET /guessing-game/list - get list of all games by their IDs
  • POST /guessing-game/ - create a new game (returns the new game and its id)
  • GET /guessing-game/:id - get a specific game by its id (e.g. /guessing-game/1 to get game with id 1)
  • POST /guessing-game/:id/player - add a player to a specific game. requires a JSON body containing name (string) and guess (number), e.g. {"name":"yossi", "guess": 85}
  • POST /guessing-game/:id/start - start the game by picking a random number; the game object is returned with the winner

What kind of data is there?

  • game is an object of the following form:

    {
    id: number,
    players: Player[],
    isOver: boolean,
    result: number | null
    winner: Player
    }

  • Player is an object of the following form:

    {
    name: string,
    guess: number,
    }
